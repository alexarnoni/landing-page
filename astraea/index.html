Funcionalidades

🔄

Pipeline de Dados

Ingestão diária via NASA NeoWs + DONKI APIs, transformação em camadas staging/mart com dbt Core, PostgreSQL como storage central.

🧠

ML Risk Scoring

Random Forest treinado em dados históricos de aproximação. Classifica cada asteroide em baixo/médio/alto risco usando 7 features orbitais.

🌐

API Pública REST

8 endpoints documentados, rate limiting de 60 req/min por IP, dados em tempo real de 1.400+ objetos. Acesse via astraea-api.alexarnoni.com/docs
